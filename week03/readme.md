## What is package.json 

The package.json file stores information about a Node project. Stores things like the project name, what version it is, the scripts and dependencies.

## What is process.env 

The process.env is how Node.js reads environment variables. These variables can be used to configure an application without changing the code directly.

## What npm start Runs

npm start runs the command that's stated in the scripts section of package.json. 

In this project it runs: node index.js

That executes the Node program.
